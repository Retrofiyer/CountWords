## Overview

This project is a simple web application built using Node.js and Express. It allows users to input a text through a web interface and calculates the total word count.

## Features

<div>
  <ul>
      <li> <b>Word Count API:</b> An API endpoint that takes a text as a query parameter and returns the word count.</li>
      <li> <b>Error Handling:</b> Validates user input to ensure text is provided, displaying an error message if not.</li>
      <li> <b>Asynchronous JavaScript:</b> Uses fetch API in the front-end to interact with the back-end asynchronously.</li>
  </ul>
</div>

## Getting Started

## Prerequisites

Before you begin, make sure you have the following tools installed on your machine:

- **Node.js 18.20.4 or higher** - [Download Node](https://nodejs.org/en/download/package-manager)

If you don't have any of these tools installed, follow the provided links to install them.


## Installation

1.- Clone the repository
   ```sh
   git clone https://github.com/Retrofiyer/CountWords.git
   cd CountWords
   ```
2.- Build project using node.js
 ```sh
   npm install
   ```
## Running the project

  ```sh
    npm start
   ```

Open any browser and type

 ```sh
   localhost:8082
   ```
